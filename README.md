# ros2_px4_stack

ACL 2024 PX4 quadcopter fleet software. 

The pixstack (ros2_px4_stack) consists of integration code for the autonomy stack as well as scripts to facilitate test flights and data collection.

To setup a new quadcopter follow this guide: https://docs.google.com/document/d/1l8QtowLM2WaQr2ADHyFJCM6RihY4-RuVmX10wdNYHUg/edit?usp=sharing
